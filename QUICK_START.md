# Quick Start Guide - ENT 101 Quiz

**For Users Without Python**

## Getting Started (3 Easy Steps)

### Step 1: Download the Application
- Get the distribution folder from your instructor
- OR run build_exe.bat if you have the source code

### Step 2: Open the Folder
- Navigate to the distribution folder
- You should see: ENT101_Quiz.exe and 4 PDF files

### Step 3: Run the Quiz
- Double-click ENT101_Quiz.exe
- The application will start immediately!

## How to Use the Quiz

### Selecting Study Mode
1. Practice Mode: Study at your own pace
2. Timed Test: 90 minutes timer
3. Review All: Browse with answers visible

### During the Quiz
- Click answer options (A, B, C, D)
- Use Previous and Next buttons
- Toggle Turkish/English explanations
- View Source Info for chapter details
- Jump to questions using number field

### Finishing
- Click Submit Quiz when done
- View your score
- Review incorrect answers

## Troubleshooting
- EXE blocked: Right-click Properties Unblock Apply
- Missing PDFs: Keep all files together
- Crashes: Restart the application
